# PyTwitchAPI

## Sobre o projeto
O **PyTwitchAPI** é o consumo da API da Twitch focada para servidores em Python

## Funcionalidades
- Autenticação da api seguindo o **Authorization Code Grant Flow**
- Criação de clips e consulta de clips
- Mensagens no Chat
- Informações de usuários
